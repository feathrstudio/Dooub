## 📂 프로젝트 개요
이 리포지토리는 본 프로젝트의 배포용 소스 코드와 구성 파일을 포함하고 있습니다.  
클라이언트 검수 및 전달을 위한 최종 버전입니다.

---

## 🛠️ 로컬 실행 방법 (Run Locally)

### 1) 패키지 설치
npm install

### 2) 환경 변수 설정  
프로젝트 루트에 `.env.local` 파일을 생성하고 아래 값을 입력해주세요:

GEMINI_API_KEY=YOUR_API_KEY

### 3) 개발 서버 실행
npm run dev

개발 서버는 기본적으로 http://localhost:3000 에서 실행됩니다.
(포트 충돌 시 자동으로 3001, 3002 등으로 변경될 수 있습니다.)
